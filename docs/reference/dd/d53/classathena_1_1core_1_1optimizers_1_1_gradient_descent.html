void athena::core::optimizers::GradientDescent::prepare ( )
overridevirtual

Generate bytecode for backpropagation

The whole algorithm:

  1. Calculate actual error E
  2. Let's Q - queue of nodes, EQ - queue of errors
    1. \( node \rightarrow Q \)
    2. \( E \rightarrow EQ \)
  3. For each node N in Q
    1. \( EQ \rightarrow E \)
    2. If this is variable node, adjust weights: \( N = N - \alpha * E \)
    3. If this is regular node, for each incoming node I:
      1. If I is constant, skip
      2. \( E_i = D_i \odot E \), where \( E_i \) is the new error value, \( D_i \) - derivative of N with respect to I, \( \odot \) - Hadamard (elementwise) product.
      3. \( E_i \rightarrow EQ \)
      4. \( I \rightarrow Q \)
