\subsection{Detailed Description}
Op\+Code specifies digital codes for Athena VM\textquotesingle{}s operations. Operation is encoded with operation code and its parameters. Bits 0..31 (assuming little-\/endian) are used by operation code. Bits 32..63 are used by operation parameters.
